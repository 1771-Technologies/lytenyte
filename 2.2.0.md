## Changes

- Initial viewport width and height:

  Two new props, `viewportInitialWidth` and `viewportInitialHeight`. These properties let the
  grid render content on first frame. Useful for SSR where data can be pre-loaded, and there
  is not DOM to use for measuring viewport dimensions.

- New `suppressScrollFlash` prop

  Suppress scroll flash will prevent the white flash whilst a user scrolls. This effectively
  makes scrolling synchronous, whilst ensuring the content is always visible even as the user
  scrolls.

- New `rowAnimate` and `columnAnimate` props

  Animate rows and columns as they move (reorder), enter (are added/shown), and exit (are
  removed/hidden). Accepts `true` for sensible defaults, or a config object to control
  `move`/`enter`/`exit` independently (duration, easing, and for enter/exit, the animation type -
  `"fade"` by default, or your own function returning custom keyframes). Set any phase to `false`
  to disable just that one.
