## Headers

Headers create created using the `#` symbol. Each `#` is one level of the header.
HTML supports 6 levels of headers, however, only the first four are styled by
in LyteNyte Doc:

```mdx
# Header 1

## Header 2

### Header 3

#### Header 4

##### Header 5

###### Header 6
```

## Text

Normal text is converted into paragraph tags. The standard MD syntax for lists,
quotes, images, and tables is also supported.

```mdx
Hello World, **BOLD**, _ITALIC_, ~STRIKETHROUGH~

1. First
2. Second
3. Third

- Item 1
- Item 2

1. First
   - Item 1
   - Item 2

2. Second
3. Third

> Quote here

![alt](https://upload.wikimedia.org/wikipedia/commons/7/70/Example.png)

| Table | Description |
| ----- | ----------- |
| Hello | World       |
```

**Unordered List**

## Auto Links

```mdx
[My Link](https://github.github.com/gfm)

This also works: https://github.github.com/gfm.
```
