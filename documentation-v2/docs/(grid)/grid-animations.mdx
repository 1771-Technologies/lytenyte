LyteNyte Grid can animate rows and columns whenever they are added, removed, or reordered.
Animations are powered by the Web Animations API and run off the React render cycle,
so they do not affect rendering performance.

Animations break down into three separate parts:

- **Enter:** The animation to play when the row or column first appears in the grid.
- **Exit:** The animation to play when the row or column leaves the grid.
- **Move:** The transition to perform when a row or column changes position.

To enable these animations you can set the `rowAnimate` or `columnAnimate` properties to:

- `true` for the default animation settings.
- Animation config, which will allow customization beyond the default behavior, and enable or disabling
  specific animation parts.

## Basic Animations

The demo below sets the `rowAnimate` and `columnAnimate` properties to `true`, which enables the
default grid animations.

You can click:

- The **Shuffle Rows** button to re-order the rows randomly.
- The **Shuffle Columns** button to re-order columns randomly.
- The **Toggle Adults** to hide all rows with the `Adult (35-64)` **Age Group**.
- The **Toggle Country** to hide the **Country** Column.
