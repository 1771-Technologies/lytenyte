## Evaluator

`Evaluator` is the entry point for the expression engine. Instantiate it once with an optional
plugin array and reuse it for all evaluations.

```ts
import { Evaluator, standardPlugins } from "@1771technologies/lytenyte-pro/expressions";

const evaluator = new Evaluator(standardPlugins);
```

      this fallback, the evaluator throws an error.
      
      Set \`undefinedIdentifierFallback\` to \`true\` in filter functions so partially typed expressions pass all rows instead

## Plugin

An object that extends one or more stages of the evaluator pipeline. Every field is optional;
implement only the hooks your plugin needs. Return `null` from any hook to signal "not handled"
and let the evaluator try the next plugin.
