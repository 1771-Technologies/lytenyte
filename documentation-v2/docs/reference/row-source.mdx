---
title: Row Source
description: This section provides a reference for the base row source type used by LyteNyte Grid.
  All row sources in the grid extend this type.
---

The `RowSource` type is an interface implemented by all grid row sources. Each row source adds
additional methods and properties specific to its behavior. When you provide a row source to
the grid, the grid exposes it through the grid API.
