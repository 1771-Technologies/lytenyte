---
title: Grid Props
description: This reference lists the props you can pass to the LyteNyte Grid component.
---

The base type for LyteNyte Grid component props is the generic `Grid.Props`, which accepts
a `GridSpec` object. The example shows the `GridSpec` interface. For a detailed
walkthrough, see the [API Extension guide](/docs/grid-api-extensions).

```ts
export interface GridSpec<
  Data = unknown,
  ColExt extends Record<string, any> = object,
  S extends RowSource<Data> = RowSource,
  Ext extends Record<string, any> = object,
> {
  readonly data?: Data;
  readonly column?: ColExt;
  readonly source?: S;
  readonly api?: Ext;
}
```
