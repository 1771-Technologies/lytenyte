---
title: Client Row Source
description: LyteNyte Grid's client row source extends the base row source with
  additional capabilities for client-side data.
---

Use the client row source when the full row dataset is available in the client browser.
See the [Client Row Overview guide](/docs/client-source-overview) for more details.

## Client Source Params

To create a client row source, use the `useClientDataSource` hook exported by LyteNyte Grid.
This hook accepts parameters that configure the client source view.

## Client Source API Additions

The client row source extends the grid API with the following additional properties.

<TypeTable
  type={[
    {
      name: "rowUpdate",
      description: "Updates a set of rows in the client data source.",
      type: "(rows: Map<RowNode, T>) => void",
    },
    {
      name: "rowDelete",
      description: "Deletes a set of rows from the client data source.",
      type: "(rows: RowNode[]) => void",
    },
    {
      name: "rowAdd",
      description: "Adds a set of rows to the client data source.",
      type: `(rows: T[], placement?: "start" | "end" | number) => void`,
    },
  ]}
/>
