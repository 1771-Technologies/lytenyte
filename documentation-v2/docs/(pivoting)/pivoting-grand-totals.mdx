## Grand Total Row

To display the grand totals row for pivots, set the `pivotGrandTotals` property on the client row data
source. The `pivotGrandTotals` property takes one of two values:

- `"top"`: The grand totals row will be pinned to the top of the viewport.
- `"bottom"`: The grand totals row will be pinned to the bottom of the viewport.

The grand totals row is an [aggregation row](/docs/row-overview). Aggregation rows summarize data but
do not contain any children. In the case of pivots the grand total row will aggregate all the rows
in the view.

The demo below shows the grand total row in action. The row is positioned at the bottom of the
viewport.

::demo[Grand Totals="./demos/pivot-grand-totals"]

The code below shows the setting on the client data source that enables the grand totals row.
To compute the grand total row aggregations the grid will use the measures provided in the `pivotModel`
property.

```ts
const ds = useClientDataSource<GridSpec>({
  data: salesData,
  pivotMode: true,
  pivotGrandTotals: "bottom", //!
  pivotModel: {
    columns: [{ id: "ageGroup" }],
    rows: [{ id: "country" }, { id: "productCategory" }],
    measures: [
      {
        dim: { id: "profit", name: "Profit", type: "number", cellRenderer: ProfitCell, width: 140 },
        fn: "sum",
      },
    ],
  },
  rowGroupDefaultExpansion: true,
  aggregateFns: { sum: aggSum },
});
```
