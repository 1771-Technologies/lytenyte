[Apache Arrow](https://arrow.apache.org/) provides the `apache-arrow` package for parsing and
creating Arrow tables. Install this package to enable exporting data to Arrow files.

<PackageInstall package="apache-arrow" />

## Download an Arrow File

The demo below demonstrates exporting data from the grid and creating and downloading
an Arrow file. Since Arrow files are binary, you will need an
[Apache Arrow file viewer](https://www.filestash.app/tools/arrow-viewer.html)
to inspect the output.

::demo[Export Arrow File="./demos/arrow-export"]

The code below shows how to create and download an Arrow table. Because the Arrow
format is columnar, the code first constructs an object of arrays, where each array
represents a column. The code then passes this object to the `tableFromArrays` export
from `apache-arrow`. Finally, the code converts the table to a byte buffer that can
be downloaded.

```tsx
<button
  data-ln-button="tertiary"
  data-ln-size="md"
  //!next 19
  onClick={async () => {
    const api = apiRef.current;
    if (!api) return;

    const rect = await api.exportData();

    const columns = Object.fromEntries(rect.columns.map((x) => [x.id, [] as any[]]));

    for (let i = 0; i < rect.data.length; i++) {
      const data = rect.data[i];

      for (let j = 0; j < rect.columns.length; j++) {
        const column = rect.columns[j];
        columns[column.id].push(data[j]);
      }
    }

    // TypeScript reports an error here, but the type is correct.
    const table = new Uint8Array(tableToIPC(tableFromArrays(columns), "file"));

    downloadBlob(new Blob([table], { type: "application/vnd.apache.arrow.file" }), "data.arrow");
  }}
>
  Download Arrow File
</button>
```
