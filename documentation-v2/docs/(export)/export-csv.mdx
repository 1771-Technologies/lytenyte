LyteNyte Grid does not include a built-in CSV parser. However, you can use LyteNyte Grid's
export functionality to generate a CSV file by parsing the output of the
[`exportData`](/docs/reference) method on the grid API.

## Download a CSV

The demo below demonstrates exporting grid data to a CSV string and then downloading
that data as a CSV file. CSV is a simple format that supports numbers and strings.

::demo[Export CSV="./demos/csv-export"]

The example below shows a basic implementation for exporting grid data to CSV.
This implementation is intentionally simple and does not handle all possible
CSV edge cases. For more robust serialization, consider using an open source
CSV library such as [csv-parser](https://github.com/mafintosh/csv-parser).

```tsx
<button
  data-ln-button="tertiary"
  data-ln-size="md"
  //!next 19
  onClick={async () => {
    const api = apiRef.current;
    if (!api) return;

    const rect = await api.exportData();

    const rows: string[] = [rect.columns.map((x) => x.name ?? x.id).join(",")];
    for (let i = 0; i < rect.data.length; i++) {
      const row: string[] = [];
      const data = rect.data[i];
      for (const x of data) {
        if (typeof x === "string") row.push(`"${x}"`);
        else row.push(String(x));
      }
      rows.push(row.join(","));
    }

    downloadBlob(new Blob([rows.join("\n")], { type: "text/csv" }), "data.csv");
  }}
>
  Download CSV File
</button>
```
