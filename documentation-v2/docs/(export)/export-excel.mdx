Several open-source libraries can generate Excel files in JavaScript.
This guide uses [ExcelJS](https://github.com/exceljs/exceljs), but
alternatives like [SheetJs](https://sheetjs.com/) work with the same approach.

## Excel Download

The demo below lets you download an Excel file containing all grid data.
It uses ExcelJS to create a `Blob` and trigger the download.

::demo[Excel Export & Download="./demos/excel-export"]

The implementation uses [`exportData`](/docs/reference/grid-api)
to retrieve the current view of cells.

```tsx
<button
  //!next 8
  onClick={async () => {
    const api = apiRef.current;
    if (!api) return;

    const rect = await api.exportData();

    downloadBlob(await getExcelFile(rect), "data.xlsx");
  }}
>
  Download Excel File
</button>
```
