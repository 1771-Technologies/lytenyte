Row banding, also known as zebra striping, alternates row background colors to improve
readability in dense tables. By default, LyteNyte Grid adds a `data-ln-alternate="true"`
attribute to every other row. Target this attribute in CSS to apply alternating styles.

## Odd Even Alternates

Set `rowAlternateAttr` to `false` to disable alternating row attributes. The demo below
shows the effect, toggle the switch off to give all rows the same background, or on to
restore the alternating pattern.

::demo[Alternating Row Stripes="./demos/row-alternating"]

:::note
`rowAlternateAttr` only controls whether LyteNyte Grid applies the `data-ln-alternate`
attribute. You still need to target it in CSS to produce visible striping.
:::

### Root Row Alternates

Passing `"root"` to `rowAlternateAttr` makes LyteNyte Grid assign the `data-ln-alternate`
value based on the root row's index rather than each individual row's index. When row groups
are present, all children in a group inherit the same `data-ln-alternate` value as their
top-level parent row.

::demo[Root Row Alternating="./demos/row-alternating-group"]
