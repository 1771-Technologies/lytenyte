Unless you've been living blissfully offline, you've probably noticed
[shadcn/ui](https://ui.shadcn.com/) taking over the React ecosystem.

Shadcn/ui is a collection of modern, accessible React components
that you can drop into your codebase or generate via a CLI. Unlike traditional UI
libraries, shadcn/ui gives you complete control over your code: every component is
easy to customize and fit with your own styles, so you
never have to compromise on your design.

At 1771 Technologies, we've been hard at work crafting something special for
the React and shadcn/ui communities. We're excited to announce that LyteNyte Grid,
our high-performance React data grid, is now live in the
[shadcn/ui registry](https://ui.shadcn.com/docs/directory).

So, if feature-rich data tables and seamless shadcn/ui integration
get you excited, you're at the right place.

## Setup LyteNyte + Shadcn/ui in 60s

LyteNyte Grid is a powerful React grid library built for massive scale,
full Tailwind compatibility, and maximum flexibility. It's the only grid
that can be headless for full control or pre-styled with one of our sleek themes.

We've now added native themes for shadcn/ui (dark and light), using
shadcn's own Tailwind token system. For developers, that means:

- No extra styling layers to manage.
- If you update your theme tokens, the grid updates automatically.
- It looks and feels like a natural extension of your shadcn/ui application.

You can install it using the standard shadcn/ui command and get up and running in
seconds. Check out our [Installation with Shadcn guide](https://www.1771technologies.com/docs/intro-installation-shadcn)
for more details, or simply run:

```tsx
npx shadcn@latest add @lytenyte/lytenyte-core
```

## Bringing Shadcn/ui to All LyteNyte Grid Users

The new Shadcn themes are part of our open-source LyteNyte Grid Core edition, which,
at only 36kb (gzipped), already offers powerful features for free, such as:

- Row grouping
- Master-detail rows
- Data aggregation

So, if you're building dashboards, admin panels, or internal tools and
want them to feel native to shadcn/ui, LyteNyte Grid handles the heavy
lifting so you can focus on features, not plumbing.
